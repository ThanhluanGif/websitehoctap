Đối tượng học viên

Phù hợp với học viên chưa từng học lập trình web hoặc mới tiếp xúc ở mức cơ bản. Học viên cần biết thao tác máy tính, dùng trình duyệt, có email cá nhân và có máy tính để thực hành.

Chuẩn đầu ra

Học viên tự tạo được website tĩnh bằng HTML/CSS, biết dùng Flexbox/Grid, làm responsive, quản lý mã nguồn bằng GitHub và triển khai website qua GitHub Pages.

Quy trình triển khai

Mỗi buổi thêm một lớp năng lực vào cùng một project

01

Khởi tạo

Tạo project, viết HTML, xác định chủ đề website và nội dung chính.

02

Thiết kế giao diện

Thêm CSS, card, hero, navbar, spacing, màu sắc và typography.

03

Dựng layout

Dùng Flexbox, Grid, position và responsive để website dùng được trên nhiều màn hình.

04

Công bố

Đưa code lên GitHub, viết README, deploy GitHub Pages và bảo vệ project.

              Rubric

              Tiêu chí đánh giá project cuối khóa

              HTML 25 điểm

              Cấu trúc đầy đủ, semantic hợp lý, nội dung rõ, link/ảnh/form hiển thị đúng, code dễ đọc.

              CSS 30 điểm

              Bố cục rõ, màu sắc dễ đọc, spacing hợp lý, dùng Flexbox/Grid đúng tình huống.

              Responsive 15 điểm

              Hiển thị tốt trên desktop và mobile, không tràn ngang, ảnh và menu dễ thao tác.

              GitHub 15 điểm

              Repository đúng cấu trúc, commit rõ, README đủ thông tin và GitHub Pages hoạt động.

              Hoàn thiện 10 điểm

              Nội dung không còn dạng nháp, sản phẩm có cá nhân hóa và các phần liên kết với nhau.

              Trình bày 5 điểm

              Demo rõ ràng, giải thích được ý tưởng, cách làm và điểm muốn cải thiện.
